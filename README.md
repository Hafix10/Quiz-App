# Quiz System (Java GUI)

# Project Overview

The Basic Online Quiz System is a Java-based desktop application developed using GUI components. The purpose of this project is to provide a simple and interactive platform where users can attempt multiple-choice questions and receive their results instantly.

# Features
User-friendly graphical user interface (GUI)
Multiple-choice question system
Automatic score calculation
Instant result display
Simple and interactive quiz experience

# Technologies Used
Java

Java Swing (GUI)

# Project Objectives
To understand GUI development in Java
To implement event handling and user interaction
To manage quiz questions and calculate scores efficiently
